# 삼백초 추천 TOP 3 검색 페이지 설계

## 목표
- 홈쇼핑 방송 상품 `코엔삼백초`를 보고 검색한 사용자를 유입시킨다.
- 유입 후에는 `삼백초 알약형` 제품 중 가성비 좋은 대안을 비교해 보여준다.
- 정적 페이지 특성에 맞게 특정 방송 편성표가 바뀌어도 오래 유지되는 구조로 만든다.

## 페이지 개요
- 파일명: `search/sambaekcho-top3.html`
- 제목: `삼백초 추천 TOP 3 - 코엔삼백초와 가성비 좋은 제품 비교`
- 포맷: `추천 + 비교 혼합형`
- 비교 대상: `삼백초 정/알약형` 제품만 사용

## 검색 의도
- 직접 유입 키워드: `코엔삼백초`
- 확장 키워드: `삼백초`, `삼백초 추천`, `삼백초 정`, `삼백초 알약`
- 기대 행동: 코엔삼백초와 비슷한 계열의 삼백초 제품을 찾고, 더 저렴한 대안을 비교해 본다.

## 핵심 설계
- 도입은 `코엔삼백초`를 직접 언급한다.
- 본문 비교는 특정 방송 상품에 종속되지 않도록 `삼백초 알약형 일반 비교`로 푼다.
- 즉, 유입은 방송 상품명으로 잡고 전환은 일반 비교 구조로 처리한다.

## 비교 기준
- `1정당 가격`
- `한 달분 체감가격`
- 보조 기준:
  - `정 수/포장 단위`
  - `알약형인지 여부`
  - `구매 접근성`

## 추천 제품 구성 원칙
- 전부 `정/알약형`만 포함한다.
- `효능 계열이 비슷해 보이는 삼백초 제품`을 우선한다.
- 쿠팡에서 검색성과 구매 전환이 가능한 상품만 후보로 사용한다.
- 구성 문법:
  - 가장 무난한 기준형 1개
  - 더 저렴한 가성비형 1개
  - 대용량 또는 정 수 기준 실속형 1개

## 페이지 구조
1. 코엔삼백초를 찾는 분께
2. 삼백초 알약형은 어떻게 비교하면 되는지
3. TOP 3 한눈에 보는 비교표
4. 제품별 추천 포인트
5. FAQ
6. 관련 건강식품 시리즈 링크용 확장 구역

## SEO 및 메타
- `title`, `description`, `canonical` 포함
- `og:image`, `twitter:image`는 전용 썸네일 사용
- `BreadcrumbList`, `ItemList`, `FAQPage` JSON-LD 포함
- `search/index.html`, `index.html`, `sitemap.xml`에 연결

## 운영 원칙
- 기준 상품 `코엔삼백초` 정보가 바뀌어도 본문 구조는 유지한다.
- 가격 비교 문구는 절대값보다 `상대 비교` 중심으로 쓴다.
- 제품 교체가 필요할 때는 `알약형`, `1정당 가격`, `한 달분 체감가격` 축을 유지한다.
