# '황금 가성비 골드픽' 리브랜딩 설계

- **날짜:** 2026-04-17
- **상태:** 초안 (사용자 리뷰 대기)

## 1. 목적

기존의 "스마트 쇼핑 큐레이션"이라는 일반적인 명칭을 **"황금 가성비 골드픽"**으로 변경하여 브랜드 정체성을 강화한다. '황금'과 '트로피' 이미지를 시각적 요소(색상, 아이콘)와 결합하여 사용자에게 신뢰감과 프리미엄 가성비라는 인상을 심어주는 것을 목표로 한다.

## 2. 핵심 변경 사항

| 항목 | 기존 (As-Is) | 변경 (To-Be) | 비고 |
|---|---|---|---|
| **브랜드명** | 스마트 쇼핑 큐레이션 | **황금 가성비 골드픽** | 사이트 제목, 헤더, 푸터 적용 |
| **메인 슬로건** | 똑똑한 소비의 시작, TOP 3 가이드 | **가성비의 정점, 황금 가성비 골드픽** | index.html 히어로 섹션 |
| **강조 색상** | 파란색 (`#2563eb`) | **골드 (`#d4a017`)** | 버튼, 링크, 포인트 요소 |
| **파비콘 (Icon)** | 장바구니 (🛒) | **황금 트로피 (🏆)** | 모든 HTML 파일 적용 |
| **메타 태그** | 스마트 쇼핑 큐레이션 | 황금 가성비 골드픽 | `og:site_name`, `<title>` 등 |

## 3. 상세 구현 계획

### 3.1 시각적 요소 업데이트 (`search/assets/style.css`)
- `--accent` 변수 값을 `#2563eb`에서 `#d4a017`로 변경하여 사이트 전반의 강조색을 골드로 통일한다.
- 필요한 경우 골드 톤과 어울리는 다크 모드/라이트 모드 대비를 점검한다.

### 3.2 HTML 메타 및 아이콘 업데이트 (전체 HTML)
- 모든 HTML 파일(`<link rel="icon">`)의 SVG/Emoji 파비콘을 🛒에서 🏆로 교체한다.
- `<title>` 태그 내의 사이트 이름 접미사를 "스마트 쇼핑 큐레이션"에서 "황금 가성비 골드픽"으로 변경한다.
- `og:site_name` 메타 태그 값을 업데이트한다.

### 3.3 콘텐츠 및 문구 업데이트
- **공통 헤더/푸터:** 모든 페이지의 `.site-title` 및 푸터 저작권 표시 문구를 변경한다.
- **index.html (메인):** 
    - 히어로 섹션(`<h1>`)을 "가성비의 정점, 황금 가성비 골드픽"으로 수정한다.
    - 설명 문구(`<p>`)를 브랜드 컨셉에 맞춰 보강한다. (예: "성능과 가격을 정밀 분석하여 엄선한 황금 가성비 아이템들을 확인하세요.")
- **claw-machine-guide.html:** 가이드 내의 쇼핑 사이트 소개 명칭을 업데이트한다.

## 4. 영향 범위 (Affected Files)
