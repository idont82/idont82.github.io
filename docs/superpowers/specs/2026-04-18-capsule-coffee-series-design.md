# 설계 문서: 캡슐커피 TOP 7 구매 가이드 시리즈 (14~16번)

본 문서는 캡슐커피 시장의 다양한 수요(머신별, 취향별, 브랜드별)를 공략하기 위해 기획된 3부작 시리즈 가이드의 설계안을 담고 있습니다.

## 1. 프로젝트 개요
- **목적:** 캡슐커피 머신 소유자들에게 최적의 캡슐을 제안하고 쿠팡 파트너스 수익 창출.
- **대상:** 네스프레소(오리지널/버츄오), 돌체구스토, 일리 머신 사용자.
- **주요 특징:** 
    - 기존 TOP 3 형식을 확장한 **TOP 7** 구성.
    - 제품 이미지를 극대화한 **카드형 요약 섹션** 도입.
    - 3개의 페이지(14, 15, 16번)를 서로 연결하는 시리즈 구조.

## 2. 시리즈 구성 계획

### 14번: [머신 중심] 내 머신에 딱! 호환 머신별 베스트 TOP 7
- **파일명:** `search/capsule-coffee-machine-top7.html`
- **전략:** "내 머신에 이 캡슐 써도 되나?"라는 의문 해결.
- **구성:** 
    - 네스프레소 오리지널 (3종)
    - 네스프레소 버츄오 (1종)
    - 돌체구스토 (2종)
    - 일리 전용 (1종)

### 15번: [취향 중심] 홈카페 완성! 맛과 용도별 추천 TOP 7
- **파일명:** `search/capsule-coffee-flavor-top7.html`
- **전략:** 머신 종류와 상관없이 '맛'의 경험을 중시하는 사용자 공략.
- **구성:** 
    - 고소한 맛 / 산미 있는 맛 / 디카페인 / 라떼용 / 가성비 / 프리미엄 / 시즌 한정 등 테마별 구성.

### 16번: [네스프레소 집중] 실패 없는 선택! 네스프레소 호환 캡슐 끝판왕 TOP 7
- **파일명:** `search/capsule-coffee-nespresso-top7.html`
- **전략:** 가장 큰 시장인 네스프레소(오리지널/버츄오) 사용자 집중 타겟팅.
- **구성:** 
    - 스타벅스, 일리, 라바짜 등 유명 브랜드의 네스프레소 호환 베스트 제품군.

## 3. UI/UX 디자인 상세

### A. 카드형 요약 섹션 (상단)
- **이미지:** 고화질 제품 패키지 + 추출된 커피 비주얼.
- **배지(Badge):** 머신 호환성(`오리지널`, `버츄오` 등) 및 특징(`가성비`, `진한 맛`)을 이미지 위에 오버레이.
- **인터랙션:** 카드 클릭 시 본문의 상세 설명 섹션으로 앵커 이동.

### B. 상세 리뷰 섹션
- **비교 지표:** 강도(Intensity), 산미, 바디감을 시각적인 바(Bar) 형태로 표시.
- **추천 대상:** "이런 분께 추천해요" 문구를 통해 구매 결정 보조.
- **링크:** 개별 쿠팡 파트너스 링크 반영 (기본 링크: `https://link.coupang.com/a/erQzkm` 활용).

## 4. 기술적 사항
- **SEO:** 각 페이지별 메타 태그 최적화 및 `sitemap.xml` 업데이트.
- **반응형:** 모바일에서 카드 섹션이 가로 스크롤 또는 2열 그리드로 최적화 노출되도록 CSS 설계.
