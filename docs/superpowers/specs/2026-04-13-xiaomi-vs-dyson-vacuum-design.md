# 샤오미 vs 다이슨 무선청소기 비교 페이지 설계

- 날짜: 2026-04-13
- 위치: `index.html`, `search/xiaomi-vs-dyson-cordless-vacuum.html`
- 상태: 승인됨

## 목적

`index.html`에 `"샤오미 vs 다이슨 — 무선청소기 뭐가 나을까?"` 카드를 추가하고, 클릭 시 이동하는 비교형 상세 페이지를 만든다.
상세 페이지의 핵심은 단순 비교표가 아니라 체크형 추천기로 사용자의 우선순위에 맞는 브랜드를 빠르게 제안하는 것이다.

## 결정 사항

- 메인 노출은 기존 카드 그리드에 새 카드 1개를 추가한다.
- 상세 페이지는 `TOP 3` 추천 글이 아니라 `vs` 비교형 에디토리얼로 만든다.
- 추천기는 체크박스 다중 선택형으로 만든다.
- 추천 톤은 균형형으로 유지하고, 점수 차가 작으면 중립 결과를 허용한다.

## 사용자 흐름

1. 메인 카드 클릭
2. 비교 페이지 진입
3. 한줄 결론 확인
4. `내게 맞는 쪽 찾기`에서 항목 체크
5. 샤오미, 다이슨, 중립 중 하나의 결과 확인
6. 아래 비교표와 설명으로 근거 확인

## `index.html` 변경

- `category-grid` 안에 새 카드 추가
- 링크: `/search/xiaomi-vs-dyson-cordless-vacuum.html`
- 설명 문구는 예산, 무게, 사용감, 관리 편의성 비교를 강조한다

## 상세 페이지 구조

1. 상단 고지
2. H1과 날짜
3. 한줄 결론
4. 체크형 추천기
5. 추천 결과 박스
6. 핵심 비교표
7. 샤오미가 맞는 경우
8. 다이슨이 맞는 경우
9. FAQ

## 추천기 설계

- 체크박스 5~7개
- 체크 즉시 결과 갱신
- 초기 상태에서는 안내 문구 표시
- 예산과 실속은 샤오미 가중치
- 가벼움과 프리미엄 체감은 다이슨 가중치
- 차이가 작으면 중립 결과 표시

## 기술 설계

- 정적 HTML + 인라인 JavaScript
- 기존 `/search/assets/style.css` 재사용
- 필요한 보강 스타일만 최소 추가

## 검증 기준

- 메인 카드에서 상세 페이지로 이동 가능
- 체크 조합에 따라 결과가 즉시 바뀜
- 초기 상태, 샤오미 우세, 다이슨 우세, 중립 상태가 모두 확인됨
- 모바일에서도 추천기와 비교표가 읽힘
