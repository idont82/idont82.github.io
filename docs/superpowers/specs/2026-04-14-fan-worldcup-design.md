# 선풍기 이상형 월드컵 혼합형 정보 페이지 설계

- 날짜: 2026-04-14
- 위치: `index.html`, `search/fan-worldcup.html`
- 상태: 승인됨

## 목적

메인 허브 페이지에 선풍기 월드컵 콘텐츠 카드를 추가하고, 클릭 시 이동하는 별도 선풍기 비교 페이지를 만든다.
이 페이지는 검색 유입용 정보 페이지이면서 동시에 `이상형 월드컵`처럼 2개 제품씩 비교하며 최종 우승 제품을 고를 수 있는 인터랙션을 제공한다.

## 결정 사항

- 페이지 성격은 `정보 페이지 + 월드컵 위젯` 혼합형으로 간다.
- 검색 의도는 `선풍기 추천`, `선풍기 비교`, `BLDC 선풍기`, `무선 선풍기` 계열을 자연스럽게 포괄한다.
- 실제 플레이는 8강 구조로 구성한다.
- 하단에는 10개 이상 후보 전체 리스트를 별도 섹션으로 제공한다.
- 각 대진 카드에는 이미지, 모델명, 핵심 기능 2개만 노출한다.
- 최종 우승 결과에는 추천 이유와 구매 링크를 함께 보여준다.

## 사용자 흐름

1. 메인 카드 클릭
2. 선풍기 월드컵 페이지 진입
3. 상단 설명으로 비교 기준 이해
4. 8강 대진에서 2개 제품 중 하나씩 선택
5. 결승 후 우승 제품 확인
6. 우승 이유와 구매 링크 확인
7. 하단 후보 전체 리스트와 FAQ로 추가 탐색

## `index.html` 변경

- `category-grid`에 새 카드 1개 추가
- 링크: `/search/fan-worldcup.html`
- 카드 제목은 선풍기 추천 월드컵 성격이 바로 드러나게 작성한다
- 카드 설명은 가성비, 바람세기, 저소음, 무선 편의성 기준으로 고르는 경험을 강조한다

## 상세 페이지 구조

1. 상단 고지
2. H1과 날짜
3. 도입부 설명
4. 선풍기 선택 기준 요약
5. 월드컵 인터랙션 영역
6. 우승 결과 박스
7. 후보 10개 이상 전체 리스트
8. 구매 가이드 섹션
9. FAQ
10. 하단 고지

## 월드컵 설계

- 8강 브래킷 기반
- 라운드마다 2개 제품 카드만 표시
- 카드에는 이미지, 모델명, 핵심 기능 2개 표시
- 제품 클릭 시 승자가 다음 라운드로 진출
- 라운드 상태 텍스트 표시 (`8강`, `4강`, `결승`)
- 마지막에는 우승 제품, 추천 이유, 구매 링크 표시
- 다시 하기 버튼 제공

## 데이터 설계

- 후보는 최소 10개 이상 준비하되 플레이용 대진은 8개 사용
- 후보 데이터는 페이지 내부 JavaScript 배열로 관리
- 각 제품 데이터는 아래 필드를 가진다
  - `id`
  - `name`
  - `image`
  - `features` (2개)
  - `summary`
  - `tags`
  - `buyUrl`

## 콘텐츠 원칙

- 월드컵은 재미 요소지만 페이지 본문은 정보형 톤 유지
- 후보 리스트는 실제 비교 검색에 도움이 되도록 텍스트 설명을 충분히 넣는다
- `가성비`, `바람세기`, `저소음`, `무선 편의성` 기준을 도입부와 하단 가이드에서 반복적으로 설명한다
- 결과는 절대 평가가 아니라 사용자의 선택 취향에 따른 추천으로 표현한다

## SEO 설계

- `title`, `meta description`, `canonical`, `robots`, Open Graph 태그 추가
- `Article`, `BreadcrumbList`, `FAQPage` 구조화 데이터 추가
- 본문에 `선풍기 추천`, `선풍기 비교`, `BLDC 선풍기`, `무선 선풍기`, `저소음 선풍기` 문맥을 자연스럽게 포함
- 후보 10개 이상 전체 리스트를 HTML에 실제 텍스트로 노출

## 기술 설계

- 정적 HTML + 인라인 JavaScript
- 기존 `/search/assets/style.css` 재사용
- 필요한 인터랙션 스타일만 페이지 내부에서 보강
- 외부 라이브러리는 추가하지 않는다

## 검증 기준

- 메인 카드에서 상세 페이지 이동 가능
- 월드컵 8강 -> 4강 -> 결승 흐름이 정상 동작
- 우승 결과와 다시 하기 버튼 동작 확인
- 후보 전체 리스트가 10개 이상 노출됨
- 모바일에서도 대진 카드와 결과 영역이 읽힘
