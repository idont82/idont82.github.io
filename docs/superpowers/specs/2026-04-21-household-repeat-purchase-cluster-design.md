# 설계 문서: 재구매형 생활소모품 클러스터 (17~19번)

본 문서는 재구매 주기가 짧고 검색 의도가 명확한 생활소모품 3개 카테고리인 화장지, 물티슈, 치약을 묶어 3부작 콘텐츠 클러스터로 확장하기 위한 설계안입니다.

## 1. 프로젝트 개요
- **목적:** 반복 구매가 많은 생활소모품 카테고리에서 검색 유입을 확보하고, 비교형·추천형 콘텐츠를 통해 쿠팡 파트너스 전환을 유도한다.
- **대상:** 가격과 성능을 함께 보는 일반 소비자, 대용량/가족용 구매자, 문제 해결형 제품 탐색 사용자.
- **핵심 전략:**
  - 상품명보다 먼저 검색되는 `비교`, `추천`, `용도`, `목적` 키워드를 우선 공략한다.
  - 3개 페이지를 서로 연결하는 생활소모품 클러스터 구조로 내부 링크를 강화한다.
  - 각 품목의 검색 의도에 맞게 포맷을 달리한다. 화장지는 가성비 비교, 물티슈는 용도별 추천, 치약은 목적별 추천으로 구성한다.

## 2. 시리즈 구성 계획

### 17번: 가성비 화장지 비교 TOP 3
- **파일명:** `search/toilet-paper-budget-top3.html`
- **검색 의도:** "대용량 화장지 뭐가 제일 무난하지?", "가격 대비 품질 괜찮은 화장지 추천"
- **핵심 비교 기준:**
  - 겹수
  - 롤 수
  - 총 길이 또는 사용 지속성
  - 먼지날림/보풀 체감
  - 무향 여부 및 자극감
  - 10m당 또는 롤당 체감 가격
- **추천 방향:** 대용량 가정용 중심으로 무난한 선택 3개를 제안한다.

### 18번: 용도별 물티슈 추천 TOP 3
- **파일명:** `search/wet-tissue-top3.html`
- **검색 의도:** "아기용 물티슈 추천", "청소용 물티슈 뭐가 좋지?", "휴대용 물티슈 추천"
- **핵심 비교 기준:**
  - 용도 적합성 (아기용/일반용/청소용)
  - 평량
  - 성분 및 저자극 포인트
  - 향 유무
  - 뚜껑형 여부/휴대성
  - 장당 가격
- **추천 방향:** 한 페이지 안에서 용도별 대표 제품 3개를 제안한다.

### 19번: 목적별 치약 추천 TOP 3
- **파일명:** `search/toothpaste-top3.html`
- **검색 의도:** "시린이 치약 추천", "미백 치약 추천", "구취용 치약 추천"
- **핵심 비교 기준:**
  - 목적 적합성 (시린이/미백/구취·데일리)
  - 기능성 성분
  - 자극감과 사용감
  - 향/맛
  - 데일리 사용 적합성
  - 가격
- **추천 방향:** 사용 목적이 뚜렷한 소비자를 위한 문제 해결형 구조로 작성한다.

## 3. 콘텐츠 구조

### A. 페이지 공통 구조
- 상단 SEO 메타 및 요약 인트로
- 카드형 또는 표형 핵심 비교 블록
- 제품별 상세 리뷰 3개
- "이런 분께 추천해요" 섹션
- FAQ 2개 이상
- 생활소모품 클러스터 내부 링크 블록

### B. 페이지별 포맷 차이
- **화장지:** 가격/용량/체감 품질이 중요하므로 `가성비 비교형`
- **물티슈:** 사용 목적 분기가 강하므로 `용도별 추천형`
- **치약:** 해결하려는 문제가 먼저이므로 `목적별 추천형`

## 4. 내부 링크 전략
- 17, 18, 19번 페이지 하단에 서로 연결되는 클러스터 네비게이션을 추가한다.
- `search/index.html`에는 세 페이지를 각각 등록한다.
- 루트 `index.html`에는 필요 시 `생활소모품 추천 시리즈` 대표 카드 1개를 추가한다.
- `sitemap.xml`에 세 URL을 모두 반영한다.

## 5. UX 원칙
- 기존 `/search/` 가이드 템플릿과 동일한 시각 언어를 유지한다.
- 비교 포인트가 명확히 드러나도록 표나 카드 정보 밀도를 조절한다.
- 모바일에서도 제품 3개 비교가 빠르게 끝나도록 길이를 과도하게 늘리지 않는다.

## 6. 구현 우선순위
1. 17번 화장지 페이지 작성 및 반영
2. 18번 물티슈 페이지 작성 및 반영
3. 19번 치약 페이지 작성 및 반영
4. 내부 링크, 허브, 사이트맵, 루트 홈 대표 카드 조정

## 7. 리스크 및 주의사항
- 화장지와 물티슈는 규격이 다양하므로 단순 가격만 비교하면 왜곡될 수 있다. 체감 단위 가격 기준을 명시해야 한다.
- 치약은 의약적 효능처럼 과장되지 않도록 `사용 목적 중심`으로 표현하고 진단/치료로 오해될 문구를 피한다.
- 물티슈는 아기용과 청소용 성격이 많이 다르므로 추천 대상을 분명히 구분한다.
