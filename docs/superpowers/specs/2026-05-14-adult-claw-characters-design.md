# 어른들이 인형뽑기에서 많이 찾는 캐릭터 허브 및 하츠네 미쿠 상세 설계

## 목표

인형뽑기 페이지에서 이미 확보한 검색 유입을 바탕으로, 어른들이 실제로 많이 찾는 캐릭터 굿즈 쪽으로 관심을 확장시키는 허브를 만든다. 최종 목적은 캐릭터별 상세 페이지에서 쿠팡 굿즈 상품으로 자연스럽게 연결되는 흐름을 만드는 것이다.

이번 범위는 두 개다.

- 허브 페이지: `어른들이 인형뽑기에서 많이 찾는 캐릭터 TOP 5`
- 1호 상세 페이지: `하츠네 미쿠 인형뽑기 굿즈 추천 TOP 5`

## 왜 이 구조로 가는가

한 페이지에서 여러 캐릭터를 모두 깊게 다루면 검색 의도와 구매 의도가 흐려진다. 반대로 허브만 만들면 쿠팡 연결이 약하다. 따라서 허브는 관심을 넓게 잡고, 상세는 한 캐릭터에 집중해서 굿즈 소비 판단을 돕는 구조가 적합하다.

이 구조의 장점은 다음과 같다.

- 허브가 `인형뽑기에서 많이 보이는 캐릭터`라는 넓은 유입을 받는다.
- 상세 페이지는 `하츠네 미쿠 굿즈`, `하츠네 미쿠 인형`, `하츠네 미쿠 피규어`처럼 더 직접적인 검색을 받는다.
- 허브에서 상세로, 상세에서 쿠팡으로 이어지는 구매 동선이 자연스럽다.

## 정보 구조

### 1. 허브 페이지

경로 제안:

- `search/adult-claw-characters-top5.html`

제목:

- `어른들이 인형뽑기에서 많이 찾는 캐릭터 TOP 5`

핵심 역할:

- 인형뽑기 경품과 잘 연결되는 어른 취향 캐릭터를 한눈에 소개
- 각 캐릭터별 굿즈 성격을 빠르게 보여주기
- 상세 페이지로 클릭을 유도

카드 구성:

1. 하츠네 미쿠
2. 치이카와
3. 쿠로미
4. 짱구
5. 스밋코구라시

각 카드에는 아래 요소를 넣는다.

- 큰 비주얼
- 한 줄 에디터 코멘트
- 굿즈 태그 3~4개
- 상세 보기 버튼

태그 예시:

- `누이`
- `피규어`
- `키링`
- `데스크 굿즈`
- `가챠 감성`

### 2. 하츠네 미쿠 상세 페이지

경로 제안:

- `search/hatsune-miku-claw-goods-top5.html`

제목:

- `하츠네 미쿠 인형뽑기 굿즈 추천 TOP 5`

핵심 역할:

- 왜 하츠네 미쿠가 어른 취향 인형뽑기 캐릭터로 강한지 설명
- 인형뽑기 경품과 잘 맞는 굿즈 타입을 정리
- 쿠팡에서 실제로 보기 좋은 굿즈 상품 5개로 연결

상품 축 예시:

1. 봉제 인형
2. 피규어
3. 키링
4. 누들스토퍼/책상 장식
5. 데스크 굿즈 또는 소형 소품

## 표현 톤

전체 톤은 `오타쿠 굿즈 매거진형`으로 간다.

의도는 다음과 같다.

- 너무 딱딱한 구매 가이드처럼 보이지 않게 한다.
- 너무 과한 팬위키 말투로 흐르지 않게 한다.
- 캐릭터를 아는 사람에게는 공감을 주고, 처음 보는 사람에게는 왜 인기 있는지 빠르게 이해시키는 글로 쓴다.

문장 스타일 기준:

- `책상 위에 두기 좋은 타입`
- `가방에 달기 쉬운 굿즈`
- `인형뽑기 경품 감성이 잘 살아나는 제품`
- `가볍게 입문하기 좋은 굿즈`

피해야 할 표현:

- 지나치게 내부 팬덤 용어만 쓰는 문장
- 운영자 시점의 SEO 설명
- `검색 의도`, `유입`, `전환` 같은 메타 표현

## 디자인 방향

### 허브

허브는 `매거진 카드형`으로 설계한다.

- 상단 히어로: 잡지 커버 같은 분위기
- 중간 카드 5개: 캐릭터별 대형 비주얼과 짧은 코멘트
- 배경: 크림/오프화이트 기반, 카드별 포인트 색 분리
- 모바일에서는 세로 카드 리스트로 자연스럽게 변환

허브 카드 예시:

- `하츠네 미쿠`
  - 코멘트: `피규어와 누들스토퍼, 키링까지 경품 감성이 가장 잘 살아나는 대표 캐릭터`
  - 태그: `피규어`, `키링`, `데스크 굿즈`

### 하츠네 미쿠 상세

- 상단 히어로 이미지: 실사풍 또는 상품 콜라주형
- 도입: 왜 어른들이 하츠네 미쿠 굿즈를 찾는지
- 비교 영역: 굿즈 타입별 추천 5개
- 하단: 관련 캐릭터 허브 또는 다음 캐릭터 상세로 연결

## SEO 방향

허브 타깃 키워드:

- `인형뽑기 캐릭터 추천`
- `어른들이 좋아하는 인형뽑기 캐릭터`
- `인형뽑기 성인 캐릭터`
- `인형뽑기 굿즈 캐릭터`

하츠네 미쿠 상세 타깃 키워드:

- `하츠네 미쿠 굿즈 추천`
- `하츠네 미쿠 인형뽑기`
- `하츠네 미쿠 피규어 추천`
- `하츠네 미쿠 키링`

메타 원칙:

- 허브는 넓은 관심 키워드 중심
- 상세는 캐릭터명 + 굿즈명 중심
- 썸네일은 `비교/추천 페이지`라는 인상이 바로 들게 구성

구조화 데이터:

- `BreadcrumbList`
- `ItemList`
- 필요 시 상세에 `FAQPage`

## 쿠팡 연결 원칙

쿠팡은 허브보다는 상세에서 더 강하게 붙인다.

허브에서는:

- 직접 상품 카드 최소화
- 상세 페이지 진입 우선

상세에서는:

- 상품 이미지
- 짧은 추천 이유
- `쿠팡에서 구매` 버튼
- 버튼 아래 수수료 고지

이 흐름이 광고 느낌을 줄이고 자연스럽다.

## 내부 링크 전략

- `claw-machine-guide.html`에서 허브로 연결 가능성 검토
- 허브에서 하츠네 미쿠 상세로 링크
- 하츠네 미쿠 상세 하단에서 허브 및 다음 캐릭터로 링크
- 검색 인덱스 및 루트 인덱스에도 카드 추가

## 범위 밖

이번 작업에 포함하지 않는다.

- 치이카와, 쿠로미, 짱구, 스밋코구라시 상세 페이지 작성
- 캐릭터별 SNS/팬덤 분석 문서화
- 별도 `character/` 섹션 신설

## 성공 기준

- 허브가 매거진형 톤으로 자연스럽게 보인다.
- 하츠네 미쿠 상세가 구매 가이드처럼 읽히되 팬덤 감성도 유지한다.
- 허브와 상세 모두 검색용 메타와 썸네일을 갖춘다.
- 상세 페이지의 쿠팡 연결이 과도하게 광고처럼 보이지 않는다.
