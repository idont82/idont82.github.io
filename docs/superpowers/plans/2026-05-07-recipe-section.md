# Recipe Section Implementation Plan

## Goal

Create a new static `recipe/` section with a hub page and two recipe detail pages: salt bread and Dubai chewy cookie.

## Steps

1. Create `recipe/index.html` as the recipe hub.
2. Create `recipe/salt-bread.html` with ingredients, steps, tips, and temporary Coupang ingredient links.
3. Create `recipe/dubai-chewy-cookie.html` with ingredients, steps, tips, and temporary Coupang ingredient links.
4. Add links from root `index.html`.
5. Add new URLs to `sitemap.xml`.
6. Verify file creation and route references.
